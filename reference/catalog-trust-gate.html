NosePrint reference card

Catalog trust gate

Five checks

  1. License chain: Is reuse permission supported by evidence?
  2. Provenance: Can each field be traced to an authorized source?
  3. Schema: Does the file have exactly the expected columns?
  4. Row count: Does the pinned file match the reviewed artifact?
  5. Quality: Have material risks been reviewed and accepted?
Rule: every check must pass. Failed or inconclusive means no Real Catalog import.

Row outcomes

Accepted → authoritative SQLite facts. Transformed → accepted with traceable cleanup. Duplicate → deterministic skip. Rejected → structurally invalid. Quarantined → questionable and excluded pending review.
