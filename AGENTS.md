## Agent skills

### Issue tracker

Issues are tracked with GitHub Issues; external PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the five default triage labels. See `docs/agents/triage-labels.md`.

### Domain docs

This is a single-context repository. See `docs/agents/domain.md`.

### Teaching after implementation

After each completed implementation issue, use `srini-personal-teacher` to explain the completed work in language a curious 12-year-old can understand.

Do not stage or commit learning-only files or folders after implementation issues unless the user explicitly asks. This includes `learning-sessions/`, `learning-records/`, `lessons/`, `reference/`, `RESOURCES.md`, `MISSION.md`, and `NOTES.md`.
