# NosePrint

NosePrint is a local-first fragrance discovery project built to make scent-comparison data and vector search inspectable.

The first implementation slice is the provenance gate for the Real Catalog. The proposed 2,191-row dataset is currently **not approved for import**. See [the audit result](docs/audits/perfume-recommendation-dataset-v1.md) and [the audit/import workflow](docs/catalog-import.md).

Run the acceptance tests with:

```bash
python3 -m unittest discover -v
```
